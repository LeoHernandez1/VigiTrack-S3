VigiTrack S3 – Sistema Inteligente de Vigilancia y Seguimiento Autónomo

Descripción del proyecto:
VigiTrack S3 es un sistema de vigilancia inteligente basado en el microcontrolador ESP32-S3, diseñado para detectar la presencia de personas mediante la combinación de sensores, activar una cámara, analizar la escena mediante inteligencia artificial y realizar seguimiento automático del objetivo utilizando servomotores. El sistema integra tanto un sensor infrarrojo pasivo (PIR) como un sensor de microondas, lo que permite mejorar la confiabilidad en la detección de movimiento. Además, envía alertas con evidencia visual al usuario a través de Telegram, permitiendo una supervisión remota eficiente. Se trata de una solución de bajo costo, autónoma y escalable, aplicable en entornos como hogares, fincas, garitas o zonas restringidas.

Flujo de trabajo y funcionamiento:
El sistema inicia en un estado de reposo, monitoreando continuamente el entorno mediante dos tipos de sensores: un sensor PIR, que detecta cambios en la radiación infrarroja asociados a cuerpos calientes, y un sensor de microondas, que detecta movimiento físico mediante efecto Doppler. Cuando cualquiera de los sensores detecta actividad, el ESP32-S3 activa la cámara y comienza la captura de imágenes. Esta detección dual permite reducir falsas alarmas y mejorar la sensibilidad del sistema. Las imágenes capturadas son procesadas mediante un modelo de inteligencia artificial que determina si hay una persona en la escena. Si la detección es positiva, el sistema calcula la posición del objeto dentro de la imagen y ajusta la orientación de la cámara mediante dos servomotores (movimiento horizontal y vertical) para mantener al objetivo centrado. Adicionalmente, el sistema puede activar una fuente de iluminación si las condiciones de luz son insuficientes. Finalmente, se envía una notificación al usuario a través de Telegram, incluyendo una imagen capturada. Si no se detecta actividad durante un tiempo determinado, el sistema retorna a su posición inicial y estado de vigilancia.

Materiales:
El sistema utiliza como componente principal un ESP32-S3, encargado del procesamiento y control general. Se integra una cámara compatible como la OV2640 para la captura de imágenes. Para la detección de movimiento se emplean dos sensores: un sensor PIR (HC-SR501) y un sensor de microondas (RCWL-0516), lo que permite una detección más robusta. El movimiento de la cámara se logra mediante dos servomotores tipo SG90 en configuración pan-tilt. Adicionalmente, se pueden incluir una fotoresistencia (LDR) para detección de condiciones de luz, un módulo relay o transistor MOSFET para controlar una fuente de iluminación, y una fuente de alimentación adecuada para todos los componentes. También se requiere una estructura mecánica que soporte la cámara y permita su movimiento.

Implementación de la inteligencia artificial:
La inteligencia artificial del sistema se basa en un modelo de detección de personas optimizado para sistemas embebidos. Este modelo proviene de TensorFlow Lite for Microcontrollers, una plataforma de código abierto diseñada para ejecutar redes neuronales en microcontroladores. El modelo ha sido previamente entrenado con grandes conjuntos de imágenes que incluyen personas en diferentes posiciones, entornos y condiciones de iluminación. Su funcionamiento consiste en recibir una imagen reducida (por ejemplo, 96x96 píxeles) y generar una probabilidad de presencia de una persona en la escena. Si esta probabilidad supera un umbral definido, el sistema interpreta que hay una persona. Adicionalmente, el modelo puede proporcionar información sobre la ubicación del objeto dentro de la imagen, lo cual permite implementar el seguimiento automático mediante servomotores. Antes de su implementación en el ESP32-S3, el modelo puede ser probado en un entorno de desarrollo en computador utilizando herramientas como OpenCV y TensorFlow en Python, lo que facilita la validación del sistema. Posteriormente, el modelo se convierte a formato .tflite y se integra en el firmware del microcontrolador. La inteligencia artificial cumple el rol de filtro avanzado dentro del sistema, trabajando en conjunto con los sensores PIR y de microondas para diferenciar entre movimiento irrelevante y presencia humana real, mejorando así la precisión y reduciendo falsas alarmas.
